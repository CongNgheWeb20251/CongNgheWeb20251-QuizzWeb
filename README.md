## âš™ï¸ Installation & Setup

### Prerequisites
- Node.js (>= 18.x)
- npm or yarn
- MongoDB

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd quizzWeb
```

### 2. Setup Backend
```bash
cd backend
npm install
npm run dev
```
or 
```bash
cd backend
yarn install
yarn dev
```

### 3. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
or 
```bash
cd frontend
yarn install
yarn dev
```
